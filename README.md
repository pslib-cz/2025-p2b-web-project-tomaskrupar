# Šimonova ruční dílna

Jedná se o prezentační web pro lokálního tvůrce dřevěných produktů, který se zaměřuje na ruční výrobu dekorací a dárkových předmětů.

## 1. Téma a popis webu
Web slouží jako online katalog a portfolio pro truhláře, který vyrábí originální dřevěné předměty. Stránka má za cíl prezentovat kvalitu řemeslného zpracování, estetiku přírodního materiálu a umožnit zájemcům prohlédnout si aktuální tvorbu.

## 2. Cíl webu
Hlavním cílem je vytvořit důvěryhodnou a vizuálně atraktivní vizitku tvůrce.
* **Informovanost:** Poskytnout základní informace o použitých materiálech a cenových relacích (formou „cena od...“).
* **Konverze:** Zprostředkovat kontakt pro individuální poptávku výroby.

## 3. Cílová skupina
Web je navržen pro:
* **Lidi hledající originální dárky:** Zákazníci, kteří preferují ruční práci před sériovou výrobou z obchodních řetězců.
* **Milovníky přírodních materiálů:** Lidé, kteří si chtějí zútulnit domov doplňky ze dřeva.
* **Návštěvníky trhů:** Lidé, kteří tvorbu znají z vánočních trhů (nebo podobných akcí) a chtějí si objednat další produkty online.

## 4. Struktura webu (Informační architektura)
Web se skládá ze tří unikátních stránek:
1. **Domovská stránka (Index):** Úvod do příběhu tvůrce, hlavní "Hero" sekce
2. **Galerie produktů:** Strukturovaný přehled výrobků (stromečky, prkénka, hračky) řešený pomocí CSS Grid.
3. **Detail produktu / Kontakt:** Podrobnější informace o možnostech zakázkové výroby a kontaktní formulář s údaji pro domluvu.
4. **Stránka O mně:** Představení řemeslníka a jeho vztahu ke dřevu. Tato stránka buduje důvěru a ukazuje autenticitu značky.


## 5. Očekávaný obsah a technologie
* **Grafika:** Návrh vytvořen v nástroji Figma.
* **Vizuály:** Fotografie produktů generované/upravené pomocí AI.
* **Kód:** Responzivní HTML5 a CSS3 (metoda Mobile-First, CSS Grid, Flexbox, CSS Layers).
